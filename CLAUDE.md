# CLAUDE.md

## About this folder

This folder contains cartographic work (QGIS) related to a doctoral dissertation, the subject of which appears to be the historical geography of eastern regions, particularly around the frontiers of the Sasanian and Arab empires.

## Inventory

### QGIS project file
- **doctorat.qgz** — main QGIS project of the dissertation.

### Vector layers (GeoPackage .gpkg)
- **centres d'étude.gpkg** — study centers.
- **fortresses.gpkg** — fortresses and fortifications.
- **frontière sassanide.gpkg** — Sasanian frontier.
- **frontière_arabe.gpkg** — Arab frontier.
- **grandes villes.gpkg** — major cities.
- **régions.gpkg** — regions.
- **montagnes.gpkg** — mountains.
- **fleuves_grandes.gpkg** — major rivers.
- **points.gpkg** — various points.
- **télégraphe.gpkg** — telegraph lines.

(The `.gpkg-shm` and `.gpkg-wal` files are SQLite auxiliary files automatically generated by QGIS.)

### KML layers
- **fleuves.kml** — rivers (Google Earth format).
- **fleuves_grandes.kml** — major rivers.
- **fleuves.qmd** / **fleuves_grandes.qmd** — QGIS metadata attached to these layers.

### Raster layers (.tif)
Several raster images at different resolutions:
- **raster(0,05).tif**, **raster(0,1).tif**, **raster(0,5).tif** — main raster at various resolutions.
- **rater(0,02).tif**, **rater(0,2).tif** — additional rasters (note: the name "rater" is likely a typo for "raster").
- **raster_próbka(0,01).tif**, **raster_próbka(0,1).tif**, **raster_próbka(1).tif** — raster samples (Polish "próbka" = sample).
- The `.aux.xml` files — attached GDAL metadata.

## Notes

- File names are mixed between French (e.g. "fleuves", "frontière", "centres d'étude", "grandes villes", "montagnes", "régions", "télégraphe") and Polish ("próbka").
- The subject of the dissertation appears to deal with the borders between the Sasanian and Arab empires, with annotated fortresses, cities, rivers, mountains, regions, and telegraph lines.
- To continue the work, the QGIS project should be opened via the **doctorat.qgz** file.
